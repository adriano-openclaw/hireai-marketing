# HireAI Samples

This directory contains approved/sample-ready HireAI creative outputs and their supporting illustrations.

## Structure

- `creatives/` — final composed social creatives. Use these as sample output references for layout, hierarchy, spacing, CTA/footer treatment, logo placement, and finished quality.
- `illustrations/` — generated illustration assets used inside sample creatives. Use these as style/subject references when generating future HireAI illustrations.
- `archives/` — zipped sample packs for one-click/raw GitHub download.

## Current sample set

### `hr-onboarding-carousel-v4`

A 4-slide carousel for HR/onboarding repetitive-question automation.

Creative lessons captured from Adriane's review:
- Preserve slides exactly when Adriane calls them perfect; Slide 2 from v3/v4 is approved/loved.
- Move the HireAI text logo bottom-left when bottom-center makes the slide feel vertically tight.
- Visually align mixed-font phrases by perceived ink/baseline, not only CSS numbers.
- Code CTAs manually if provided SVG assets have broken embedded artwork.
- Put footer copy inside a real footer container.
- Lower/adjust illustrations when they overlap critical text.
