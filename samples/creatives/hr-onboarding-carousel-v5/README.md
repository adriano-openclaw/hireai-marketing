# HR Onboarding Carousel v5 — Approved Ground Truth

Final approved/sample-ready HireAI HR onboarding carousel.

Adriane approved this version as perfect. Use it as concrete ground truth for future HireAI creative generation, not just loose inspiration.

- `1080/` contains Discord preview-size PNGs.
- `2160/` contains high-quality source/download PNGs.

Key reusable lessons:
- Bottom-left HireAI text logo is preferred when bottom-center feels vertically tight.
- Mixed-font inline text must be visually aligned by perceived ink/baseline.
- Card text must be visually centered vertically, not only numerically centered.
- Use supplied working CTA PNG assets when available.
- Footer should be a full-width soft bottom band with centered `Let someone handle the rest.` and centered orange `https://hireai.bot`.
- Keep illustrations clear of important words/highlights.
