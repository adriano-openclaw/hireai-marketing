## Reusable layout rules

- Canvas: 1080×1080 square creative with white/light background and subtle cool-gray dotted pattern.
- Brand chrome: top-left exact HireAI box logo; top-right 4-dot pagination with orange active dot; bottom wordmark when layout calls for it.
- Headline: large black high-contrast serif type, usually in the upper third/half.
- Emphasis: orange handwritten/script words or phrases, often underlined, used sparingly inside or near the headline.
- Body/support copy: bold black sans-serif for main explanatory copy; lighter gray sans-serif for secondary lines.
- Illustration: black-outline business/sales editorial illustration with orange accents; usually lower half or right side, often backed by a pale orange blob/oval.
- CTA: dark rounded rectangle with white text and subtle orange accent/shadow; usually lower-left when used.
- Spacing: generous safe margins; keep top chrome clear; balance text-heavy areas with illustration mass.

## Common layout templates

1. Text-left / illustration-right / CTA lower-left.
2. Centered headline / centered illustration / wordmark footer.
3. Headline + bullets upper half / large illustration lower half / wordmark bottom-left.
4. Headline upper third / illustration middle / support copy bottom.
